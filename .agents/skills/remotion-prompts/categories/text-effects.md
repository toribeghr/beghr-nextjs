# Text Effects

Prompts focused on typography and text animations.

## Prompts in this category

Currently, text effects are primarily featured as part of image animation prompts. Check out:

- [News Screenshot Animation](../prompts/news-screenshot-animation/) - Features text highlighting with rough.js
- [Arabic News Animation](../prompts/arabic-news-animation/) - Includes RTL text handling and multi-target highlighting

## Common Text Effects

- **Highlighting**: Animated highlighter effects behind text
- **Typewriter**: Character-by-character text reveal
- **Fade in/out**: Smooth text transitions
- **Kinetic typography**: Text that moves and animates
- **Text tracking**: Adjusting letter spacing dynamically

## Tools & Libraries

Common libraries used for text effects in Remotion:

- **rough.js**: Hand-drawn, sketchy text decorations
- **Framer Motion**: Smooth text animations
- **GSAP**: Advanced text animation sequencing
- **Remotion's spring**: Physics-based text motion
