# Image Animation

Prompts for animating static images with effects like zoom, rotation, blur, and highlighting.

## Prompts in this category

### [News Screenshot Animation](../prompts/news-screenshot-animation/)
Animates a news screenshot with OCR, subtle zoom, 3D rotation, blur effects, and highlighted text.

**Key features:**
- OCR text detection with Tesseract
- 3D rotation effects
- Blur transitions
- Rough.js highlighter animations
- Customizable highlight targets

**Author:** [@Remotion](https://x.com/Remotion)

---

### [Arabic News Animation](../prompts/arabic-news-animation/)
Similar to News Screenshot Animation but optimized for Arabic text with structured parameters.

**Key features:**
- Arabic language support
- Precise rotation angles (±7.5° Y, ±3° X)
- Specific zoom values (1.0 → 1.05)
- Multiple highlight targets
- Custom blend modes

**Author:** [@Abdullah4AI](https://x.com/Abdullah4AI)

### [Maps Animation](../prompts/maps-animation/)
Create dynamic map animations with camera movements, 3D buildings, and data overlays using Mapbox.

**Key features:**
- Mapbox integration for high-quality maps
- 360-degree camera rotation
- Zoom and fly transitions between locations
- 3D building rendering
- Lower third data overlays

## Common Use Cases

- News and media content
- Social media posts
- Educational content
- Infographics
- Article highlights
- Quote animations
- Travel and location showcases
- Geographic data visualization
- Real estate and venue tours

## Typical Effects

- **Zoom**: Subtle scale animations to add depth
- **Rotation**: 3D rotation for dynamic feel
- **Blur**: Reveal effects with blur transitions
- **Highlighting**: Emphasize key text or elements
- **OCR**: Automatic text detection for precise positioning
- **Camera movements**: Cinematic fly-throughs and orbits
- **3D rendering**: Buildings, terrain, and geographic features
- **Data overlays**: Lower thirds, labels, and statistics

## Getting Started

All image animation prompts require:
1. Remotion skill installation
2. Static image file
3. Basic understanding of Remotion composition

Most also benefit from:
- Tesseract OCR for text detection
- rough.js for hand-drawn style effects
