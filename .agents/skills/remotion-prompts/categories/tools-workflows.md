# Tools & Workflows

Prompts for building tools, workflows, and meta-applications that enhance video creation.

## Prompts in this category

### [Video Editor Interface](../prompts/video-editor-interface/)
Create a canvas-based video editing interface using the Remotion skill.

**Key features:**
- Visual canvas interface
- Scene-by-scene editing
- Asset upload and management
- Iterative workflow
- Browser-based or local

**Author:** [@rileybrown](https://x.com/rileybrown)

## Integration Opportunities

Remotion workflows can integrate with:
- **MCP Servers**: For external APIs and services
- **CI/CD**: Automated video generation in pipelines
- **CMS**: Dynamic video from content
- **Design Tools**: Import from Figma, Sketch, etc.
- **Analytics**: Data-driven video generation

## Meta-Applications

These prompts create tools that make video creation easier:
- Video editors built on Remotion
- Template generators
- Batch processing scripts
- Preview systems
- Rendering farms

## Advanced Concepts

- **"Ultrathink"**: Encouraging deep research before implementation
- **Openness**: Design for easy modification
- **Iterative**: Enable changes after initial generation
- **Composability**: Build reusable components and systems
