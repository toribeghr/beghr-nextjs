# Remotion Prompts

A curated collection of community-submitted prompts for creating videos with [Remotion](https://www.remotion.dev/) and Claude Code.

## What is this?

This repository collects high-quality prompts that leverage the Remotion skill in Claude Code to generate impressive video animations. From news screenshot animations to full product demos, these prompts showcase what's possible when combining Remotion's programmatic video capabilities with AI assistance.

## Quick Start

1. Install the Remotion skill in Claude Code:
   ```bash
   npx skills add remotion-dev/skills
   ```

2. Browse the [prompts](#prompts) below and find one that fits your needs

3. Copy the prompt and use it in Claude Code with your own content

4. Customize and iterate on the generated video
